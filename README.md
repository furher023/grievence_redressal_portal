To Start:-


 STEP 1:- Fork this repo into your account.
 
 
 STEP 2:- Clone this app on your local system.
 
 
 STEP 3:- Open your command terminal window:-
 
 
 STEP 4:- Make sure you are in cloned repo.
 
 
 STEP 5:- Install all dependencies
 ```
 npm install
 ```
 STEP 6:- To run this
 ```
 npm start
 ```
 OR
 ```
 npm run dev
